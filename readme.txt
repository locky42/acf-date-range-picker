= Compatibility =

This ACF field type is compatible with:
* ACF 5
* ACF 4

== Installation ==

1. Copy the `acf-date-range-picker` folder into your `wp-content/plugins` folder
2. Activate the Date Range Picker plugin via the plugins admin page
3. Create a new field via ACF and select the Date Range Picker type
4. Read the description above for usage instructions
